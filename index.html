Hello, Builders!

This is a website that can be used during the duration of the Builder's Bootcamp to test your microservices. There are three services that will be tested: a url shortener, a batch url shortener, and a bookmark service. For each service, you can add your endpoint for the API you have defined for your microservice, and you can test with different inputs and see the API response returned by your API!
